# 프로젝트 설명서 (project-context.md)

현재 우리가 만들고 있는 **KimchiRun** 게임에 대한 전체적인 설계도와 다름없는 파일입니다.
집을 지을 때 설계도가 필요하듯, 어떤 게임을 만들고 있는지 전체적인 내용을 여기에 정리해두면 AI가 상황을 더 잘 파악하고 똑똑하게 코드를 짜줄 수 있습니다.

## 🎮 게임 개요
* **프로젝트 이름:** KimchiRun (김치 런)
* **장르:** 2D 플랫포머 / 런닝 게임
* **사용 엔진:** Unity 2D

## 🏃 플레이어 (Player)
* 플레이어는 화면을 달리며 장애물을 피하거나 아이템을 먹어야 합니다.
* **등장 개념:** 중력(Gravity)과 충돌(Collision), 사용자의 키보드 입력(Input) 등을 다룹니다.
* **핵심 스크립트:** `Assets/Scripts/Player.cs` (현재 작업 중)

## 📌 주요 특징 (기획 중)
* **물리 엔진 활용:** Unity의 `Rigidbody2D`를 사용하여 플레이어가 진짜 공중으로 붕 뜨고 떨어지는 듯한 자연스러운 점프를 구현하게 될 것입니다.
* 앞만 보고 달리는 런닝 게임의 특성상 배경이 뒤로 움직여서 마치 앞으로 달리는 것처럼 착각하게 만드는 기법이 쓰일 수 있습니다.
